## 🚀 恭喜！如果你能在网站上看到这就说明全通了！

这是一篇通过 **Obsidian** 本地编写，利用 **Git** 自动同步到 **宝塔面板** 并构建发布的测试文章。

### 🛠️ 这个流程是如何工作的？

1.  我在本地 **Obsidian** 写下这段文字。
2.  通过 **Git** 推送到 GitHub。
3.  GitHub 告诉 **宝塔面板**："嘿，有新货了！"
4.  宝塔面板自动拉取代码，执行 `hexo g` 生成静态网页。
5.  最终呈现在你面前。

### 💻 代码块测试

来测试一下代码高亮功能是否正常：

```bash
# 这是一个自动部署的伪代码
function deploy() {
    echo "正在从 GitHub 拉取..."
    git pull origin master
    echo "正在构建静态页面..."
    hexo generate
    echo "部署完成！"
}
