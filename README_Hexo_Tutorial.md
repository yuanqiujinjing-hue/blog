# Hexo + Fluid 主题博客使用与部署教程

## 1. 本地启动与常用命令

在 `my-blog` 目录下，您可以使用以下命令来管理您的博客。

### 启动本地预览
这是您最常用的命令。启动后，可以在浏览器访问 `http://localhost:4000` 查看效果。
```bash
npx hexo server
# 或者简写
npx hexo s
```
*按 `Ctrl + C` 停止服务。*

### 清理与生成
当您修改了配置或感觉样式没有更新时，建议先清理再生成。
```bash
# 清理缓存 (删除 public 目录和 db.json)
npx hexo clean

# 生成静态页面 (生成 public 目录)
npx hexo generate
# 或者简写
npx hexo g
```

## 2. 如何添加新文章

### 创建文章
使用以下命令创建一篇新文章。文章文件会生成在 `source/_posts/` 目录下。
```bash
npx hexo new "我的第一篇文章"
```

### 编辑文章
找到 `source/_posts/我的第一篇文章.md`，使用 Markdown 编辑器（如 VS Code, Typora）打开。

文件头部是 **Front-matter**，用于定义文章属性：
```yaml
---
title: 我的第一篇文章
date: 2026-02-03 10:00:00
tags: [生活, 记录]     # 标签
categories: [日常]     # 分类
index_img: /img/example.jpg  # (可选) 文章首页缩略图
banner_img: /img/banner.jpg  # (可选) 文章页顶部大图
---
```
在 `---` 下方即可开始撰写正文内容。

## 3. 部署到宝塔面板 (公网部署)

Hexo 是静态博客，部署非常简单。您只需要将生成的 HTML 文件上传到服务器即可。

### 第一步：本地生成静态文件
在您的电脑上，进入 `my-blog` 目录，执行生成命令：
```bash
npx hexo clean
npx hexo generate
```
执行完毕后，您会发现目录下多了一个 `public` 文件夹。这里面的所有内容就是您的完整网站。

### 第二步：宝塔面板操作

1.  **登录宝塔面板**：进入“网站” -> “PHP项目” (虽然是静态站，通常也在网站管理里添加) -> “添加站点”。
2.  **填写信息**：
    *   **域名**：填写您的域名（如果没有域名，填写服务器IP）。
    *   **根目录**：选择一个目录，例如 `/www/wwwroot/my-blog`。
    *   **PHP版本**：选择“纯静态” (Static)。
3.  **上传文件**：
    *   点击刚创建网站的“根目录”链接，进入文件管理。
    *   将本地 `my-blog/public/` 文件夹内的 **所有文件** (包括 index.html, css, js 等) 打包压缩。
    *   在宝塔文件管理中上传该压缩包并解压。
    *   确保 `index.html` 在网站根目录下，而不是在子文件夹里。

### 第三步：访问测试
在浏览器输入您的域名或服务器IP，即可看到博客已经上线。

### (进阶) 如何更新网站？
每次发布新文章时：
1.  本地运行 `npx hexo clean && npx hexo g`。
2.  重新上传 `public` 目录下的内容覆盖服务器文件。
*注：也可以配置 Git 自动化部署，但上述手动上传方式最简单直观。*
